# KG-AgriBot 

Knowledge Graph-based Conversational AI for Crop Disease Diagnosis

## Features
- EfficientNet-B3 based disease classification
- Knowledge graph-based reasoning
- Confidence-aware dialogue system
- Explainable AI outputs

## Dataset
- PlantVillage (Corn subset)
- 4 classes: Blight, Rust, Cercospora, Healthy

Classes:
- Cercospora Leaf Spot
- Common Rust
- Northern Leaf Blight
- Healthy
